## Key Rules

- Agents do not own domain entities: they receive `{Agent}Command`, return `{Agent}Result`, and persist nothing directly.
- Every agent follows the fixed pattern: validate command → load data → build envelope → call Gemini → validate structured output → log execution → return result.
- Prompts are versioned file-based StringTemplate (`.st`) files in `src/main/resources/prompts/` — never inline prompts in Java code.
- The REST API is internal only (service-to-service OIDC auth, not public).
- Every run produces a structured `AgentExecutionLog` record.
- Agents generate and suggest; they never finalize scores, silently modify approved rubrics, or deliver output to students without teacher approval.
