Status / category pill. Use tones to map to evaluation states — e.g. `success` = corregida, `warning` = en revisión, `info` = en curso, `neutral` = borrador.

```jsx
<Badge tone="success" dot>Corregida</Badge>
<Badge tone="brand">Cerrada</Badge>
<Badge tone="gold">Global</Badge>
```

Tones: neutral, brand, gold, success, warning, danger, info, solid. Optional `dot` for live status.
