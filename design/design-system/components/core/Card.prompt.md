Surface container. Compose with the static slots `Card.Header`, `Card.Title`, `Card.Body`, `Card.Footer`.

```jsx
<Card interactive accent>
  <Card.Header><Card.Title>Prueba Unidad 3 · Biología</Card.Title><Badge tone="success">Corregida</Badge></Card.Header>
  <Card.Body>32 estudiantes · promedio 5,4</Card.Body>
  <Card.Footer><Button size="sm">Ver resultados</Button></Card.Footer>
</Card>
```

`elevation`: flat / default / raised. `interactive` adds hover-lift. `accent` adds a brand top bar.
