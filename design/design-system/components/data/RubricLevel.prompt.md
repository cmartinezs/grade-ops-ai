Performance-level badge for rubric-graded open assessments. Each rubric index resolves to one ladder step.

```jsx
<RubricLevel level="exceeds" />   {/* Logrado destacado */}
<RubricLevel level="developing" />{/* En desarrollo */}
<RubricLevel level="notmet" solid />
```
Levels: exceeds, meets, developing, beginning, notmet. `solid` for dense heatmap cells.
