Determinate progress bar for bulk AI grading, file uploads, and course/assessment completion.

```jsx
<ProgressBar value={18} max={32} label="Corrigiendo con IA" showValue striped />
<ProgressBar value={84} tone="success" label="Cobertura del curso" showValue />
```
Tones: brand / gold / success / danger. `striped` animates for live jobs.
