# Epic 08 — Teacher Report

## Narrative

**As a** teacher who has completed an assessment cycle,  
**I want** a structured report summarizing outcomes, learning gaps, recommended next teaching actions, and time saved,  
**so that** I can understand what happened, communicate results, and archive evidence of the cycle.

## Goal

Produce the final human-readable artifact of the open-assessment pipeline: a teacher report that consolidates approved grading, feedback, and gap analysis into a single deliverable. The report also serves as hackathon evidence of AI-native operations.

## Scope

**In scope**
- Teacher Report Agent generating a structured report from approved grading, feedback, and gap data
- Report content: assessment summary, submission count, score distribution/stats, learning gaps, recommended next teaching action, time-saved estimate
- Export as downloadable or copyable text (P1)

**Out of scope**
- Automated report delivery to external systems or LMS integrations
- Student-visible report or report shared directly with parents/administration
- Scheduled or recurring report generation

## Epic Acceptance Criteria

- Teacher Report Agent generates a complete report once grading and gap analysis are finalized.
- Report includes: assessment summary, submission count, score distribution or key stats, learning gaps, recommended next teaching action, and time-saved estimate.
- Report is generated by the Teacher Report Agent and the run is logged.
- (P1) Teacher can export or copy the report; the export does not expose internal agent prompts.
- Export includes only teacher-approved outputs.

## Dependencies

| Epic | Reason |

| Epic 05 — Grading Assistance | Report requires finalized grading data |
| Epic 06 — Feedback Generation and Approval | Report references approved feedback outcomes |
| Epic 07 — Learning Gaps and Recovery | Report includes gap summary and recovery recommendations |

## Definition of Done

- US-070 passes all acceptance criteria.
- Teacher Report Agent is called server-side; report is persisted and retrievable.
- Agent execution log is created for the report generation run.
- Report content reflects only teacher-approved grading and feedback.
- Time-saved estimate is clearly labeled as an estimate.
